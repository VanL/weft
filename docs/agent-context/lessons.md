# Lessons Pointer

Durable project lessons are recorded in:

- `docs/lessons.md`

Use this file as the stable pointer from agent-context docs. Do not duplicate
the full lesson list here; keep one canonical lessons ledger to avoid drift.

## When To Add A Lesson

- A correction exposed a repeated failure mode.
- A workflow edge case caused preventable rework.
- A runbook or checklist needed to become stricter.

## Required Pairing

- Add the lesson in `docs/lessons.md`.
- If it is broadly reusable, add or update a runbook in `runbooks/`.
